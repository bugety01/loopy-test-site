# Loopy Test — Apple-inspired Product Ad Website

정적 웹사이트 예제입니다. Apple 공식 사이트가 아니라, 프리미엄 제품 광고형 랜딩페이지 스타일을 연습하기 위한 콘셉트 페이지입니다.

## 파일

- `index.html` — 페이지 구조
- `styles.css` — 반응형 비주얼/디바이스 mockup/애니메이션 스타일
- `script.js` — scroll reveal, header glass 효과, hero device parallax

## 실행

브라우저에서 `index.html`을 직접 열면 됩니다.

PowerShell에서 간단 서버를 쓰려면:

```powershell
cd "$HOME\Documents\loopy test"
python -m http.server 4173
```

그 다음 브라우저에서:

```text
http://127.0.0.1:4173
```
